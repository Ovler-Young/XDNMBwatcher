# Originally a separate project, this is now a part of the main project

Forked from [inkrss-web](https://github.com/pureink/inkrss-web)

## Setup

1. Install [Node.js](https://nodejs.org/en/download/)

2. Install pnpm

    ```bash
    npm install -g pnpm
    ```

3. Install dependencies

    ```bash
    pnpm install
    ```

4. Build

    ```bash
    pnpm run build
    ```
